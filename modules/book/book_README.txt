book - 圖書管理
       圖書查詢、借閱管理

程式來自
       校園自由軟體交流網 ( http://sfs.wpes.tcc.edu.tw )

作者
       book1.0 -- HaMi( cik@mail.wpes.tcc.edu.tw )      

安裝
       	將程式檔解開
       	修改 book_config.php 設定
       	
說明   
        1.本程式需合學籍系統才可運作
        2.PHP 需加上 GD 模組才能印出條碼
	3.授權學生操作權限 設定
	進入 系統管理 > 學務程式設定 > 使用權管理 
	授權管理選項 點選 "授權管理「圖書管理」的人員設定 "
